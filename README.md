# 🚀 Arkad Performance - Premium Landing Page

Bem-vindo ao repositório oficial da **Arkad Performance**, uma landing page de alta conversão focada em Gestão de Tráfego, Business Intelligence e Automação para operações industriais, concessionárias e B2B de alto ticket.

![Design Preview](https://github.com/gsguimaraes1/lp-gabriel/blob/main/public/assets/logo-arkad.png?raw=true)

## 💎 Diferenciais do Projeto

Esta LP foi construída com foco em **autoridade** e **experiência do usuário (UX)**, utilizando tecnologias de ponta para criar um ambiente digital imersivo.

*   **Loading Screen Premium**: Interface de pré-carregamento animada (0-100%) com transição fluida `Power4.out`.
*   **Interactive 3D Background**: Elementos flutuantes em tempo real utilizando `Three.js` e `React Three Fiber`.
*   **Custom Cursor System**: Cursor inteligente com detecção de hover e exibição dinâmica de labels.
*   **GSAP Animations**: Reveal de elementos e efeitos magnéticos em botões via ScrollTrigger.
*   **Data-Driven UI**: Dashboard de performance embutido e visualização de stack tecnológica (GTM, Meta, GA4, n8n, Power BI).

## 🛠️ Tech Stack

*   **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
*   **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animações**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
*   **3D Engine**: [Three.js](https://threejs.org/) (@react-three/fiber & @react-three/drei)
*   **Ícones**: [Lucide React](https://lucide.dev/)

## 🚀 Como Executar

1.  Clone o repositório:
    ```bash
    git clone https://github.com/gsguimaraes1/lp-gabriel.git
    ```
2.  Instale as dependências:
    ```bash
    npm install
    ```
3.  Inicie o servidor de desenvolvimento:
    ```bash
    npm run dev
    ```
